## 變量

### 宣告式

```
吾有[N][type]。曰[value1]。曰[value2]...曰[valueN]。名之曰[name1]曰[name2]....[nameN]
```

### 例

| 文言 | JS |
|---|---|
| `吾有一數。曰三。名之曰「甲」。` | `var a = 3;` |
| `有數五十。名之曰「大衍」。` | `var dayan = 50;` |
| `吾有一言。曰「「噫吁戲」」。名之曰「乙」。` | `var b = "alas!";` |
| `吾有一爻。曰陰。名之曰「丙」。` | `var c = false;` |
| `吾有一列。名之曰「丁」。` | `var d = [];` |
| `吾有三數。曰一。曰三。曰五。名之曰「甲」曰「乙」曰「丙」。` | `var a=1,b=3,c=5;` |

### 簡寫

```wenyan
有數五十。名之曰「大衍」。
// 同：吾有一數。曰五十。名之曰「大衍」。
```

### 重賦值

```wenyan
昔之「甲」者。今「乙」是矣。    // a = b（塊外用矣）
昔之「甲」者。今「乙」是也。    // a = b（若/若非塊內用也結塊）
```

### 引用賦值

```wenyan
吾有一數。曰「甲」。名之曰「乙」。   // var b = a;
```

### 型詳

| 關鍵字 | 型 | 字面量例 |
|---------|------|-----------------|
| `數` | number | `三`, `一百二十`, `三千二百零一`, `二〇二〇`, `三點一四` |
| `言` | string | `「「text」」`, `『text』` |
| `爻` | boolean | `陽`(true), `陰`(false) |
| `列` | array | 宣空，`充`填 |
| `物` | object | `其物如是...是謂...之物也` |
| `元` | auto | 編譯器推斷 |

`今有`→`var name = this.name = value`（導出/公）。
`吾有`→私有/域內。

## 流程控制

### 條件

| 文言 | JS |
|---|---|
| `若三大於二者。乃得「「想當然耳」」也。` | `if (3>2){ return "of course"; }` |
| `若三不大於五者。乃得「「想當然耳」」。若非。乃得「「怪哉」」也。` | `if(3<=5){return "of course"}else{return "no way"}` |

比較符：
- `大於`→>
- `不大於`→<=
- `小於`→<
- `不小於`→>=
- `等於`→==
- `不等於`→!=

塊結構：
```
若「x」大於「y」者。
  ⋯⋯
也。
```

若/若非：
```
若「x」大於「y」者。
  ⋯⋯
若非。
  ⋯⋯
也。
```

### 循環

| 文言 | JS |
|---|---|
| `為是百遍。⋯⋯云云。` | `for (var i = 0; i < 100; i++){ ... }` |
| `恆為是。⋯⋯云云。` | `while (true) { ... }` |
| `凡「天地」中之「人」。⋯⋯云云。` | `for (var human of world){ ... }` |
| `乃止。` | `break;` |

循環`云云`結，條件`也`結。

## 容器

| `夫「甲」之一。` | `a[0]` |

| `夫「玫瑰」之「「名」」。` | `rose["name"]` |

## 物

| 文言 | JS |
|---|---|
| `吾有一物。名之曰「甲」。` | `var a = {};` |
| `吾有一物。名之曰「甲」。其物如是。物之「「乙」」者。數曰三。物之「「丙」」者。言曰「「丁」」。是謂「甲」之物也。` | `var a = {b:3, c:"d"}` |

## 術

### 定義（無參）

```wenyan
吾有一術。名之曰「吸星大法」。是術曰。
  ⋯⋯
是謂「吸星大法」之術也。
```

### 定義（有參）

```wenyan
吾有一術。名之曰「翻倍」。欲行是術。必先得一數。曰「甲」。乃行是術曰。
  乘「甲」以二。名之曰「乙」。
  乃得「乙」。
是謂「翻倍」之術也。
```

多參帶型：
```wenyan
吾有一術。名之曰「fn」。欲行是術。必先得一數曰「a」。二言。曰「b」。曰「c」。乃行是術曰。
  ⋯⋯
是謂「fn」之術也。
// function fn(number a, string b, string c)
```

### 調用

```wenyan
施「翻倍」於「大衍」。    // double(dayan)
```

### 返回

| 文言 | JS |
|---|---|
| `乃得四十九` | `return 49;` |
| `減五十以一。乃得矣` | `return 50-1;` |
| `乃歸空無` | `return;` |

### 自動柯里化

術皆自動柯里化：
```wenyan
施「f」於一於二。名之曰「半f」。    // var halfF = f(1)(2)
施「半f」於三。書之。              // console.log(halfF(3))
```

## 棧式嵌套調用

`夫`入棧。`取N以施「fn」`出N項調fn，結果入棧。

```wenyan
注曰「「result = g(a, f(b, c))」」
夫「a」。夫「b」。夫「c」。取二以施「f」。取二以施「g」。名之曰「result」。
```

複雜嵌套：
```wenyan
注曰「「j = i(f(e(a,d(b,c))),g,h)」」
夫「a」。夫「b」。夫「c」。取二以施「d」。取二以施「e」。取一以施「f」。夫「g」。夫「h」。取三以施「i」。名之曰「j」。
```

算術結果及列項亦可入棧：
```wenyan
注曰「「result = f(a+b, arr[3], arr.length)」」
加「a」以「b」。夫「arr」之三。夫「arr」之長。取三以施「f」。名之曰「result」。
```

棧特殊變量：
- `其`→最近無名結果；兼清棧
- `噫`→棄/清無名棧

## 錯誤處理

### 試/捕

```wenyan
姑妄行此。                                    // try {
  嗚呼。「「滅頂」」之禍。曰「「覆巢之下」」      //   throw {name:"滅頂", message:"覆巢之下"}
如事不諧。豈「「虛指」」之禍歟。名之曰「e」。     // } catch(e) if ReferenceError {
  夫「e」之「「名」」。書之。                    //   console.log(e.name)
豈「「文法」」之禍歟。                          // } catch if SyntaxError {
  ⋯⋯
豈「「異類」」之禍歟。                          // } catch if TypeError {
  ⋯⋯
不知何禍歟。名之曰「e」。                       // } catch(e) {（全捕）
  ⋯⋯
乃作罷。                                      // }
```

### 拋出

```wenyan
嗚呼。「「ErrorName」」之禍。曰「「error message」」
// throw new Error，帶名帶訊息
```

訊息可省：`嗚呼。「「ErrorName」」之禍。`

### 內建錯誤型

| 文言 | JS |
|---|---|
| `文法` | SyntaxError |
| `逾界` | RangeError |
| `異類` | TypeError |
| `虛指` | ReferenceError |

### 簡式試/捕

```wenyan
姑妄行此。
  ⋯⋯
如事不諧乃作罷。
// try { ... } catch(_) {}
```

## 巨集

`或云`/`蓋謂`定替代語法：

```wenyan
或云「「書「甲」焉」」。
蓋謂「「吾有一言。曰「甲」。書之」」。

書「「問天地好在」」焉。
// 展開為：吾有一言。曰「「問天地好在」」。書之。
```

參數用天干：`「甲」「乙」「丙」「丁」「戊」「己」「庚」「辛」「壬」「癸」`（至多10）。

巨集不可以標識符/參數起止。

標準庫`列經`用巨集美化高階函數：
```wenyan
凡「某列」皆「加一」其上者。書之。
// 展開為：施「遍施」於「加一」於「某列」。書之。
```

## 注釋

```wenyan
批曰。「「注釋」」。
注曰。「「注釋」」。
疏曰。「「注釋」」。
```

皆編譯為JS之`/* comment */`。

## 輸出

```wenyan
吾有一數。曰五。書之。    // console.log(5)
```

`書之`印棧上所有無名變量。
